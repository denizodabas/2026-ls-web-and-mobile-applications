# 📱 Mobile App & Web Development: Erasmus Brew (2026)

Welcome to the Mobile App & Web Development course. This semester, we are moving beyond simple coding to focus on the **Lifecycle of a Digital Product**. You are the manager of "Erasmus Brew," a digital coffee shop, and your goal is to build, secure, optimize, and market this brand.

---

## 📋 Course Essentials
* **Format:** 1.5h Practical Seminar followed by a 0.75h Lecture.
* **Requirement for Credit:**
    * **Attendance:** Minimum 80% (Maximum 2 missed sessions).
    * **Final Project:** Submission of the Final Report via GitHub Pull Request.
    * **Final Review:** One-on-one session in Week 12.
* **Grading:** Pass/Fail (Credit based).

---

## 📝 Final Submission Requirements (Week 11-12)
To receive credit for the course, you must submit a **Final Report** documented in this repository and your repository as well.

### 🛠 Submission Steps
1.  **Format:** A single Markdown (`.md`) file od a PDF file created in MS Word.
2.  **Location 1:** `your-repository/week-11/REPORT.pdf`.
3.  **Location 2:** `my-repository/week-11/reports/NAME_SURNAME_REPORT.pdf`.
4.  **Deadline:** All code and reports must be pushed to GitHub before the Week 12 session.
5.  **Method:** **Pull Request (PR)**.
    * PR Title: `[ODD/EVEN] Final Report - Surname Firstname`

### 📝 Report Structure
Write about what you did, what you learned. Use the notes and images, that you have created and saved over the 10 weeks in your repository. Have fun :)
